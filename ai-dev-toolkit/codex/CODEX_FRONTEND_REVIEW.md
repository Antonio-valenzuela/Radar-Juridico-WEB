# Prompt y checklist: frontend

```text
Revisa el frontend de <ALCANCE> sin rediseñarlo automáticamente. Evalúa arquitectura de componentes, contratos API, UX, accesibilidad, responsive y rendimiento. Si existe URL local autorizada, verifica visualmente los caminos críticos.

Entrega hallazgos con ruta/componente, impacto para usuario, severidad y corrección concreta. Separa defectos de preferencia estética.
```

- [ ] HTML semántico, headings, labels y landmarks
- [ ] Teclado, foco visible, modales y lectores de pantalla
- [ ] Contraste, zoom, texto escalable y reduced motion
- [ ] Desktop, móvil, orientación y touch targets
- [ ] Loading, empty, error, offline y permission denied
- [ ] Server/client boundaries y estado sin duplicación
- [ ] Formularios con validación accesible y prevención de doble envío
- [ ] Bundle, imágenes, fuentes, waterfalls y cache
- [ ] Core Web Vitals y rutas críticas
- [ ] XSS, contenido externo y datos sensibles en cliente
