# Workflow: auditoría web full-stack

## Frontend

- Rutas, render server/client, estado y contratos API.
- Semántica, teclado, foco, contraste, responsive y reduced motion.
- Estados loading/empty/error/denied y recuperación.
- Bundle, waterfalls, caché, imágenes y Core Web Vitals.

## Backend

- Matriz de rutas pública/autenticada/admin/interna.
- Validación, autorización por recurso, errores y rate limits.
- Servicios cohesionados, transacciones, idempotencia y timeouts.
- APIs externas con retries acotados y circuit breaking cuando proceda.

## Integración

- Tipos compartidos o contrato versionado.
- Propagación de correlation ID y errores seguros.
- E2E de caminos críticos y permisos negativos.
- Deploy compatible entre frontend, API, workers y esquema.
